Schedule Hawk Application

The purpose of this application it to provide a GUI to manage appointments and customers on and existing production
database for global consulting organization.

Author: Michael Bros
Email: mbros@wgu.edu
Version: 1.0
Publish Date: 2022/04/25

IDE: IntelliJ IDEA 2021.3.2 (Community Edition)
JDK: zsrc17.30.15-jdk17.0.1
JavaFX: JavaFX-SDK-17.0.1
MySQL Connector Driver: mysql-connector-java-8.0.25

Open the project using the IDE and libraries listed above. Then run the ScheduleHawkApplication.java class. Make sure
you are in an environment where it has access to the database as defined in DBConnection.java.
